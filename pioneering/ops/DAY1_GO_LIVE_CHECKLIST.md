# Day-1 Go-Live Checklist

## Pre-Launch (60 min)
- [ ] Pick final plan names and prices (Lite/Pro/Business)
- [ ] Upload 3 ZIP files to your selling platform (e.g., Gumroad)
- [ ] Paste listing copy from `products/whatsapp-reply-kit/publish/GUMROAD_LISTING.md`
- [ ] Add one cover image (use text from `THUMBNAIL_COPY.md`)
- [ ] Set product category + tags
- [ ] Enable instant delivery/download
- [ ] Add support contact in listing

## Launch (90 min)
- [ ] Publish listing
- [ ] Post launch copy (Arabic + English) from `LAUNCH_POSTS.md`
- [ ] Send first 30 DMs using `OUTREACH_DM.md`
- [ ] Log every outreach row in `SALES_TRACKER_DAY1.csv`

## Follow-up (same day)
- [ ] Follow up after 4–6 hours for all “seen/no reply” leads
- [ ] Send sample snippet to interested leads
- [ ] Update status and notes in tracker

## End-of-day Review
- [ ] Count: leads reached / interested / paid
- [ ] Identify top objection and update DM script
- [ ] Prepare Day-2 follow-up list
